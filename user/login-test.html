🚀 用户端统一登录接入测试

📋 接入说明
用户端已成功接入统一登录系统：
  • 用户端地址：http://localhost:3000
  • 统一登录地址：http://localhost:9527/login
  • 用户类型：1 (普通用户)
🔧 功能特性
  • ✅ URL参数自动接收（id, token）
  • ✅ 用户类型验证（只允许userType=1）
  • ✅ 自动清除URL参数保持干净
  • ✅ localStorage持久化存储
  • ✅ API请求自动携带用户ID
  • ✅ 统一退出到登录页面
🔗 登录流程示例
当用户从统一登录页面跳转时，URL格式如下：
http://localhost:3000/?id=USER123&token=eyJhbGciOiJIUzI1NiIs...
应用会自动：
  1. 提取URL参数中的id和token
  2. 存储到localStorage中
  3. 清除URL参数
  4. 初始化用户状态
🔐 打开统一登录 👤 打开用户端
📝 测试步骤
  1. 点击"打开统一登录"，进入登录页面
  2. 选择"用户"角色，输入登录信息
  3. 登录成功后会自动跳转到用户端
  4. 验证用户信息是否正确显示
  5. 测试各个功能模块的访问权限
